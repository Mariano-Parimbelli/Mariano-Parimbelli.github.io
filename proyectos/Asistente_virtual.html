Ikaro: Asistente Virtual para Windows

Manejado por voz

Descripción del Proyecto

Ikaro es un asistente virtual diseñado para el sistema operativo Windows que se controla completamente mediante comandos de voz. Este proyecto tiene como objetivo facilitar la vida de las personas con discapacidades o dificultades para utilizar el mouse o el teclado, brindándoles una interfaz intuitiva y eficiente.

Funcionalidades Principales

  • Control parcial del sistema operativo mediante comandos de voz.
  • Ejecución de aplicaciones y programas específicos.
  • Gestión de archivos y carpetas.
  • Navegación web utilizando comandos de voz.
  • Interacción con servicios en línea populares, como correo electrónico y redes sociales.

Tecnologías Utilizadas

Ikaro está desarrollado principalmente en Python y utiliza técnicas de inteligencia artificial para reconocimiento de voz. La interfaz minimalista está creada con Tkinter para ofrecer una experiencia limpia y fácil de usar.
