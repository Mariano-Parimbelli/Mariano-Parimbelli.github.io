Modelo de clasificación de sentimientos de tweets

Pprocesamiento del lenguaje natural (PLN)

Descripción del Proyecto

Se trata de un modelo de aprendizaje automático para clasificar el sentimiento de los tweets. Utiliza técnicas de NLP y aprendizaje automático para analizar el texto y determinar si es positivo, negativo o neutral.

Funcionalidades Principales

  • El modelo analiza unigramas y bigramas para extraer características del texto de los tweets y clasifica los sentimientos con un clasificador de árbol de decisión. Los resultados se visualizan con una matriz de confusión y una gráfica de barras.

Tecnologías Utilizadas

Programado en Python, el uso de NLP y clasificadores de árbol de decisión sugiere que se podrían haber utilizado librerías de Python como NLTK o scikit-learn para el procesamiento de lenguaje natural y el aprendizaje automático.
