Clasificación de imágenes Botellas de Plástico / Vidrios

con reconocimiento de objetos

Descripción del Proyecto

Descripción del Proyecto: El proyecto se centra en la clasificación de imágenes de botellas para identificar si son de plástico o de vidrio. Se utiliza un dataset artificial para entrenar el modelo de aprendizaje automático..

Funcionalidades Principales

  • Incluye la preparación de imágenes con transformaciones como escalar, inclinar, hacer zoom e invertir. Además, se define un modelo de red neuronal convolucional con diferentes funciones de activación para el entrenamiento y evaluación.

Tecnologías Utilizadas

Todo escrito en python, uso de algoritmos de aprendizaje automático y la plataforma Kaggle para el dataset. También se utilizan generadores de entrenamiento y validación para procesar las imágenes.
